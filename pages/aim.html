784 Studios Instant Messenger Service

784 Studios has an experimental AIM server you can connect to whenever available. The service is used for live text communication between viewers and broadcasters during live airing events. Connection information below. By using this service, you agree to the IM Service Terms and Conditions.

  1. Download an AIM client. We reccomend version 5.1
  2. On login screen, click on settings, then connection
  3. Change host to aim.784studios.net and port to 5190
  4. Press the Enter key twice to save and exit settings
  5. Login with your registered ScreenName and password

Contact admin@784studios.net to register or report an issue. If your issue is software related and is beyond our conprehensable knowledge, please try reporting an issue to the RAS GitHub repository instead.
